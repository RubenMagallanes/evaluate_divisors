Odd number of divisors problem:

Given 3 inputs A, B and K:
where 1 < A < B
and K is a positive odd number.
Implement the function evaluateDivisors(a,b,k) in the file scripts.js
The function should return the number of integers between A and B inclusive that have exactly K divisors.

The solution must pass the jasmine unit tests provided, the results of which are shown on this page. When in 'live preview' mode the tests will re-run automatically when plunker detects changes in the code. It maybe helpful to turn on and off and on live preview to stop this while you are coding.

Once you solution is running, try unskipping the tests for larger ranges, to evaluate how efficient your solution is. A well thought-out and optimised solution will be able to run these large ranges in a few seconds.
